# edge.org Random Conversation

Simple hack project to load conversations from https://www.edge.org and pick from them at random.

Uses [Alpine.js](https://alpinejs.dev/) & [daisyUI](https://daisyui.com/).

[https://carl-underwood.github.io/edge-org-random-conversation/](https://carl-underwood.github.io/edge-org-random-conversation/)

## Loading latest conversations

Run `npm run load-conversations`.
